## 🕋 프로젝트 구조

### 🗂 폴더 구조

#### backend

```bash
├──
├── modules     # 여러 모듈들을 다룬다.
├── config        # AWS, Database 등 개인 정보를 다룬다.
├── apis          # REST API와 middleware을 다룬다.
├── models        # 데이터베이스 모델
├── services      # 기능을 다룬다.
└── loaders        # 서버 첫 시작 시 로드되어야 할 파일을 다룬다.

```
